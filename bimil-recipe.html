비밀레시피(인스타그램 봇) 개인정보 처리방침

시행일: 2026년 7월 9일

본 애플리케이션(이하 "앱")은 인스타그램 계정 @bimil.recipe 운영자가 자신의 계정을 관리하기 위해 사용하는 도구입니다. 앱은 Meta 플랫폼 API를 통해 운영자 본인 계정의 콘텐츠 게시, 댓글 확인, 메시지 응답을 자동화합니다. 앱은 개인정보 보호법 등 관련 법령과 Meta 플랫폼 정책을 준수합니다.

1. 수집·처리하는 정보

  • 댓글 정보: 운영자 계정 게시물에 작성된 댓글의 내용과 작성 시각 — 키워드('레시피') 포함 여부 확인 목적으로만 처리
  • 메시지 정보: 운영자 계정과 사용자 간 다이렉트 메시지(DM) 내용 — 레시피 안내 자동 응답 목적으로만 처리
  • 공개 프로필 정보: 메시지를 보낸 사용자의 사용자명, 팔로우 여부 — 응답 대상 확인 목적으로만 조회

2. 이용 목적

  • 게시물 댓글에 대한 자동 안내 답글 및 DM 발송
  • 요청한 사용자에게 요리 레시피 정보 전달

3. 보관 및 파기

  • 앱은 처리 상태 기록(응답 완료 여부, 메시지 처리 시각)만 운영자의 로컬 장비에 보관합니다.
  • 메시지·댓글 원문, 사용자 프로필 정보는 별도 서버나 데이터베이스에 저장하지 않습니다.
  • 처리 기록은 운영 종료 시 즉시 파기됩니다.

4. 제3자 제공

앱은 수집한 정보를 제3자에게 제공, 판매, 공유하지 않습니다.

5. 데이터 삭제 요청

본인의 데이터 처리 기록 삭제를 원하시면 데이터 삭제 안내를 참고하시거나 아래 연락처로 요청해 주세요. 요청 접수 후 지체 없이 삭제합니다.
